## 📊 Что задокументировано?

### 10 API методов

**GET методы (5):**
- `getStudentGroups()` - Загрузить группы
- `getStudentInfo()` - Информация студента
- `getStudentAttendance()` - Посещаемость
- `getStudentProgress()` - Прогресс обучения
- `getStudentNotes()` - Заметки

**POST методы (5):**
- `setAttendanceMark()` - Отметить явку
- `setTrainerPresence()` - Отметить тренера
- `changeStudentGroup()` - Переводить группу
- `updateStudentInfo()` - Обновить информацию
- `createStudentNote()` - Создать заметку

### 3 модальных окна

1. **AttendanceModal** (id: `attendance`)
   - Отмечает явку студента на занятии
   - Отображает историю изменений

2. **AttendanceStatusModal** (id: `attendance-status`)
   - Статус явки

3. **TrainerPresenceModal** (id: `trainer-presence`)
   - Отмечает присутствие тренера

### 1 основной компонент

- **GroupsTab.vue**
  - Отображает все группы студента
  - Таблица занятий
  - Статистика посещений
  - История групп
